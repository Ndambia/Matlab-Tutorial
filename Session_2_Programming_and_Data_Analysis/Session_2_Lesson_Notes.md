# Session 2 Lesson Notes
## Conditionals, Loops, and Engineering Data Analysis
**Duration:** 3 hours

## Learning Objectives
By the end of this session, students should be able to:
- use logical conditions in MATLAB
- automate repeated engineering calculations
- import data from a CSV file
- calculate basic statistics
- visualize and interpret measured data

## Part 1: Conditional Statements
Teach:
- `if`
- `elseif`
- `else`
- `switch`

Example applications:
- classify a battery voltage level
- determine whether a sensor reading is safe
- choose an action based on a menu option

## Part 2: Loops
Teach:
- `for` loop for fixed repetitions
- `while` loop for condition-based repetition

Example applications:
- current calculation across multiple resistors
- repeated signal generation
- threshold search
- filling arrays

## Part 3: Data Import
Explain:
- what a CSV file is
- why engineers log data
- how MATLAB reads data using `readtable`

## Part 4: Statistics and Visualization
Teach:
- mean
- minimum
- maximum
- standard deviation
- moving average idea
- plotting one or more signals together

## Engineering Context
Use examples like:
- temperature sensor data
- vibration amplitude
- motor speed measurements
- voltage variation over time

## In-Class Questions
1. When is `while` better than `for`?
2. Why do measured engineering signals contain noise?
3. What does the mean tell you?
4. Why is smoothing useful?

## Wrap-Up
Students should see MATLAB as a tool for:
- automation
- monitoring
- experimental analysis
- report preparation
